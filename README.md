<!-- GETTING STARTED -->

### Getting Started

1. Clone the repo

```sh
  git clone https://github.com/SilasRodrigues19/nlw-esports-backend.git && cd nlw-esports-backend
```

2. Install NPM packages

```sh
  npm install
```

3. Run the project

```sh
  npm run dev
```

4. Run Prisma Studio

```sh
  npx prisma studio
```
